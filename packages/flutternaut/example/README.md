# Flutternaut Example App

A Flutter app with 5 screens demonstrating all Flutternaut widgets and `@FlutternautView` annotations.

For a full walkthrough, see the [Getting Started guide](https://flutternaut.app/docs).

## Screens

- **Login** — inputs, buttons, error text
- **Home** — todo list with indexed items, checkboxes, delete buttons
- **Gestures** — scroll, swipe, long press, double tap, drag & drop
- **Device** — orientation display
- **Control Flow** — counter, conditionals, delayed elements, numbered list, long scroll

## Test Data

The `testdata/` folder contains 5 ready-to-run test files covering all actions. Import them into the [Flutternaut desktop app](https://flutternaut.app/download) to run them on an emulator or physical device.
